class chiptools.common.colourer.ColouredStreamHandler(stream=None)[source]

Bases: logging.StreamHandler

Use this class in place of a logging.StreamHandler if coloured log output is desired. The ColouredStreamHandler will not output coloured text on Windows platforms if colorama is not installed

static add_colour(msg, levelname)[source]
emit(record)[source]

Emit a record.

If a formatter is specified, it is used to format the record. The record is then written to the stream with a trailing newline. If exception information is present, it is formatted using traceback.print_exception and appended to the stream. If the stream has an ‘encoding’ attribute, it is used to determine how to do the output to the stream.
